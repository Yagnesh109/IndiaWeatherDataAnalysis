# India Weather Data Analysis & Clustering

Performed large-scale weather data analysis on 1.15 million+ records from 365 Indian cities using Python, Jupyter Notebook, and machine learning techniques. Conducted data cleaning, visualization, and K-Means clustering to identify temperature patterns, weather trends, wind energy potential, and environmental insights across multiple regions.

Key Features:

* Exploratory Data Analysis (EDA) on weather datasets
* Interactive visualizations for temperature, humidity, pressure, wind speed, and cloud cover
* K-Means clustering for temperature pattern discovery
* Wind energy potential assessment and environmental trend analysis
* Data-driven insights for forecasting, renewable energy, and urban planning
